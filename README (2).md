# Conversor de Moedas (BRL ⇄ USD ⇄ EUR)

Conversor de moedas simples, feito em HTML, CSS e JavaScript puro. Converte valores entre Real, Dólar e Euro, com opção de inverter as moedas selecionadas.

## Funcionalidades

- Conversão entre BRL, USD e EUR em qualquer direção
- Botão para inverter as moedas selecionadas
- Exibição da taxa de câmbio utilizada
- Interface responsiva

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Como usar

Basta abrir o arquivo `index.html` no navegador. Não há dependências externas.
